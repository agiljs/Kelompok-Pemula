PENGERTIAN WEB

Web (singkatan dari World Wide Web atau WWW) adalah sistem informasi yang terdiri dari berbagai
halaman atau dokumen yang saling terhubung dan dapat diakses melalui internet menggunakan
protokol HTTP (HyperText Transfer Protocol). Web memungkinkan pengguna untuk mengakses berbagai
informasi, aplikasi,dan layanan di internet dengan menggunakan perangkat seperti komputer, ponsel, atau tablet.


Web terdiri dari elemen-elemen utama sebagai berikut:


1. Halaman Web (Web Pages): Dokumen yang berisi teks, gambar, video, dan elemen lainnya yang dapat dilihat melalui browser. Halaman-halaman ini saling terhubung melalui hyperlink.

2. Website: Kumpulan halaman web yang saling terkait dan biasanya dikelola oleh individu, organisasi, atau perusahaan. Website bisa berisi berbagai jenis konten, seperti informasi, layanan, atau aplikasi.

3. Browser: Aplikasi yang digunakan untuk mengakses dan menampilkan halaman web, seperti Google Chrome, Mozilla Firefox, Safari, atau Microsoft Edge.

4. URL (Uniform Resource Locator): Alamat unik yang digunakan untuk mengidentifikasi halaman web atau file di internet, seperti https://www.example.com.

5. Server Web: Komputer atau sistem yang menyimpan dan mengirimkan halaman web ke pengguna melalui browser mereka.
